# Logo Dateien

## Ablageort
Lege dein offizielles Golden Geometry Logo hier ab.

## Benennung
- **Logo-Datei**: `logo.svg` (empfohlen für beste Qualität) oder `logo.png`
- **Favicon**: `favicon.svg` oder `favicon.png` (falls separat)

## Format
- **SVG** (empfohlen): Skalierbar, scharfe Darstellung in allen Größen
- **PNG**: Mindestens 512x512px für hohe Qualität

## Verwendung
Das Logo wird verwendet in:
- Navbar (Header)
- Footer
- Loader-Animation
- Favicon (optional)

## Dateinamen
- `logo.svg` oder `logo.png` - Hauptlogo
- `logo-white.svg` oder `logo-white.png` - Weiße Version (falls benötigt)
- `logo-dark.svg` oder `logo-dark.png` - Dunkle Version (falls benötigt)
